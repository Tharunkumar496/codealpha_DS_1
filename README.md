# codealpha_DS_1
TASK 1: Iris Flower Classification  Use measurements of Iris flowers (setosa, versicolor, virginica) as input data.  Train a machine learning model to classify the species based on these measuremen  Use libraries like Scikit-learn for easy dataset access and model building.  Evaluate the model's accuracy and performance using test data.

## 📌 Project Overview
The goal of this project is to perform Exploratory Data Analysis (EDA) on the Iris dataset to understand the characteristics of different iris flower species.
The analysis includes:
* Loading the dataset
* Viewing dataset structure
* Data cleaning and inspection
* Summary statistics
* Species distribution analysis
* Data visualization
* Basic insights from the dataset

## 📊 Dataset
The project uses the **Iris Dataset**, which contains measurements of iris flowers from three different species:
* Iris Setosa
* Iris Versicolor
* Iris Virginica
Features:
* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species
